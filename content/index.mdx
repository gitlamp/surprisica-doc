Surprisica is a python package based on [Surprise](http://surpriselib.com/) for building and analyzing context-aware recommender systems.
It is actually a scikit package including some extended similarity measures and hybrid models extracted from some best practices in
building CARS.

** This package is specifically designed to support location-based recommender systems.
Please note that it does not support implicit ratings and content-based information.**

Any kind of feedback or critism would be greatly appreciated. Please feel free to check the [Github page](https://github.com/gitlamp/Surprisica) of package.
